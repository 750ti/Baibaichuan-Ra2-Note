

# 百百川的红警笔记

**我是百百川**，这是我的**红警笔记**仓库!  
在做视频的时候，有很多内容太啰嗦，就不方便**做进视频里**，  
所以，跟视频相关的补充内容和说明，我会写在这个红警笔记仓库里面。  

比如视频提到了 `红警地图编辑器`，随之而来的问题 就是 **下载安装和使用** 等等的问题，  
所以我会直接写在本仓库中，并贴出相关链接，方便观众、和红警玩家们，也方便自己。  
仓库的文章内容，我会陆陆续续的编写和更新，敬请期待！
